# Flask图书管理系统源码分享

## 核心功能
- 用户借阅、归还图书
- 管理员进行图书管理
- 可视化图书信息

## 适用场景
- 学校图书馆管理
- 企业图书室管理
- 个人图书管理

## 技术特点
- 使用Python编程语言
- 基于Flask框架
- 采用MySQL数据库存储数据
